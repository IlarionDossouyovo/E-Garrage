## 🛠️ Technologies

- **Backend:** Node.js, Express.js
- **Base de données:** SQLite (JSON file-based)
- **Authentification:** JWT avec refresh tokens
- **IA:** OpenAI GPT-4 API
- **Paiements:** Stripe, PayPal
- **Frontend:** HTML5, CSS3, JavaScript vanilla
- **Design:** Custom CSS avec variables néon

## 📦 Installation

```bash
# Clonez le projet
git clone https://github.com/IlarionDossouyovo/E-Garrage.git

# Allez dans le dossier
cd E-Garrage

# Installez les dépendances
npm install

# Lancez le serveur
PORT=12000 node server.js
```
