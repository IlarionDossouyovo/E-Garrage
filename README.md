# E-Garrage - E-Commerce & Dropshipping Platform

A complete e-commerce and dropshipping platform for managing online stores, products, suppliers, and automated order fulfillment.

## Features

### Basic (Free)
- Product catalog management
- Shopping cart
- Basic checkout
- User authentication

### Premium
- Advanced analytics dashboard
- Multi-supplier dropshipping automation
- Inventory sync system
- Order tracking automation
- VIP customer support
- Marketing tools
- API access

## Tech Stack
- Backend: Node.js + Express
- Database: SQLite
- Frontend: HTML/CSS/JavaScript
- Deployment: Ready for cloud hosting

## Installation
```bash
npm install
npm start
```
